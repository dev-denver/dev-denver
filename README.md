## 👨‍💻 About Me

> 비즈니스와 도메인에 대한 이해를 바탕으로 문제를 해결하는 개발자입니다.

## 🤖 AI Assisted Development

AI를 단순히 코드를 작성하는 도구로 사용하지 않습니다.

- 반복적인 작업 자동화
- 다양한 해결 방법 탐색
- 코드 리뷰 및 개선 아이디어 도출
- 설계 검토 및 리팩터링
- 결과를 직접 검증하고 개선

AI의 결과를 그대로 사용하는 것이 아니라, **직접 검증하고 비즈니스 요구사항에 맞게 개선하는 과정**을 중요하게 생각합니다.
